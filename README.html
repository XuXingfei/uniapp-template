项目文档 ​

这是 uni-app-vue3-template 项目的文档站点，基于 VitePress 构建。

目录结构 ​

docs/
├─.vitepress/    # VitePress 配置
├─guide/         # 使用指南
├─images/        # 文档图片
├─index.md       # 首页
└─README.md      # 此文件

开发命令 ​

bash
# 启动开发服务器
npm run docs:dev

# 构建文档站点
npm run docs:build

# 预览构建结果
npm run docs:preview
